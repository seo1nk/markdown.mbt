Markdown literal renderer

Source-preserving rendering: the visible text equals the lossless serialization of the document, so the rendered HTML can be overlaid on a syntax-highlighted source view glyph-for-glyph. Markers (#, **, `, list bullets, fence ticks, blockquote >) are wrapped in <span class="md-marker" aria-hidden="true"> so screen readers skip them.
